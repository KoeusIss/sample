## Installation

```elixir
@deps [
database: "~> 0.1.0"
]
```

Then:

1. Update `config/{dev,test,prod}.exs` with your database credentials.

2. `mix deps.get`

3. `mix ecto.gen.migration` is needed

4. Update the schema in lib/database/table.ex

5. Write your API in lib/database.ex
